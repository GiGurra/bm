# bm

[![CI Status](https://github.com/GiGurra/bm/actions/workflows/ci.yml/badge.svg)](https://github.com/GiGurra/bm/actions/workflows/ci.yml)
[![Go Report Card](https://goreportcard.com/badge/github.com/GiGurra/bm)](https://goreportcard.com/report/github.com/GiGurra/bm)
[![Docs](https://img.shields.io/badge/docs-gigurra.github.io%2Fbm-blue)](https://gigurra.github.io/bm/)

Searchable bookmark database CLI with text (FTS5) and semantic (Ollama) search.

```bash
go install github.com/GiGurra/bm@latest
bm import && bm fetch && bm index
bm search "golang tutorials"
```
